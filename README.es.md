# Haz estos 3 pasos y ya estás en vivo 🚀

**Solo toma 3 minutos.**<br>
Después de esto no necesitás volver a tocar GitHub nunca más.

</div>

---

<details>
<summary><h2>Paso 1 — Activá tu página</h2></summary>

> 📹 *GIF: Clic en pestaña Configuración → Páginas en el menú lateral → Desplegable de Rama seleccionar "main" → Guardar*

1. Hacé clic en la pestaña **Configuración** en la parte superior del repositorio
2. En el menú lateral izquierdo, hacé clic en **Páginas**
3. Bajo **Rama**, seleccioná **main** → hacé clic en **Guardar**

Tu página estará activa en ~2 min en:

```
https://TU-USUARIO.github.io/music/
```

</details>

---

<details>
<summary><h2>Paso 2 — Creá tu token de acceso</h2></summary>

> 📹 *GIF: github.com/settings/tokens/new → escribir "Unmuted" → Sin vencimiento → tildar repositorio → Generar token → copiar*

1. Entrá a **[github.com/settings/tokens/new](https://github.com/settings/tokens/new)**
2. **Nota:** escribí `Unmuted`
3. **Vencimiento:** elegí `Sin vencimiento`
4. Bajá y tildá la casilla **repositorio**
5. Hacé clic en <img src="https://img.shields.io/badge/Generar%20token-238636?style=flat-square&logoColor=white" alt="Generar token"/>
6. **Copiá el token** — no lo vas a ver de nuevo después de salir de esa página

</details>

---

<details>
<summary><h2>Paso 3 — Subí tu música 🎵</h2></summary>

> 📹 *GIF: abrir URL del panel → pegar token → escribir usuario → Conectar → panel carga*

1. Abrí tu página en el navegador:
   ```
   https://TU-USUARIO.github.io/music/
   ```
2. Pegá tu token, escribí tu nombre de usuario de GitHub, hacé clic en **Conectar**
3. Subí tu música y presioná **Guardar cambios** — ¡ya estás en vivo! 🎉

</details>
