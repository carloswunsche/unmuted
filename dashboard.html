Connect your GitHub

To save changes to your artist page, you need a GitHub Personal Access Token. It's free and takes about 60 seconds to create.

1
Go to github.com/settings/tokens/new
2
Give it a name like "Unmuted" and set expiration to No expiration
3
Under repo, tick the checkbox — that's the only permission needed
4
Click Generate token and copy the token that appears
5
Paste it below 👇
Stored only in your browser. Never sent anywhere except GitHub.
